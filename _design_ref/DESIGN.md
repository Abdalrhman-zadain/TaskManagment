# Design System Documentation

## Theme Overview

This document outlines the core visual characteristics of our design system, reflecting the current theme settings.

## Color Palette

Our color palette is designed to create a consistent and harmonious visual experience.

-   **Color Mode**: `light`
-   **Primary Color**: `#1275e2` - This vibrant blue is used for primary actions, key interactive elements, and branding.
-   **Secondary Color**: `#5f78a3` - A supportive, muted blue-grey, ideal for less prominent UI elements, secondary buttons, and chips.
-   **Tertiary Color**: `#c55b00` - An accent orange, utilized for highlights, badges, or decorative elements to draw attention.
-   **Neutral Color**: `#74777f` - A foundational grey for backgrounds, surfaces, and non-chromatic components.

## Typography

A clear and readable typography system is essential for effective communication.

-   **Headline Font**: `inter` - Used for all headings and titles, providing a modern and legible look.
-   **Body Font**: `inter` - Applied to all paragraph text and general content, ensuring readability across devices.
-   **Label Font**: `inter` - Utilized for form labels, small text, and UI annotations, maintaining consistency with other text elements.

## Shape and Form

The roundedness defines the character of our UI components.

-   **Roundedness**: `2` - A moderate level of roundedness, providing a balanced and friendly aesthetic.

## Spacing

Our spacing system ensures visual hierarchy and readability.

-   **Spacing**: `2` - A normal level of whitespace, offering a good balance between density and spaciousness, ensuring comfortable visual separation.
