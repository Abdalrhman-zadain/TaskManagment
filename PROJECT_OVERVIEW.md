# TeamTask — Project Overview

TeamTask is a full-stack web application designed for team and task management within organizations. It streamlines collaboration, task assignment, and progress tracking for different user roles.

## Key Features
- **User Roles:** CEO, Manager, Employee (role-based permissions)
- **Authentication:** Secure login system
- **Task Management:** Create, assign, and track tasks
- **Section & Score Management:** Organize work and monitor performance
- **Notifications:** Stay updated on important events
- **Modern UI:** Built with React and Tailwind CSS
- **RESTful API:** Backend powered by Node.js, Express, and Prisma
- **Database:** Data stored in PostgreSQL

## Purpose
TeamTask helps companies efficiently manage teams, delegate work, and monitor progress in a centralized platform.
